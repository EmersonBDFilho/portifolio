Meus Projetos

Projeto 1

Desenvolvi um site que explora as realidades das diferentes classes sociais, utilizando a técnica de paralaxe para criar uma experiência visualmente envolvente e interativa. A ideia do projeto é mostrar como as desigualdades sociais se manifestam em diversos aspectos da vida cotidiana, como educação, saúde e trabalho, e como esses fatores variam entre as classes.

Projeto 2

Eu e meu amigo Pedro Lara criamos um site para conscientizar sobre os impactos do uso excessivo das redes digitais na saúde mental. O site oferece informações sobre os efeitos desse comportamento e inclui um formulário interativo para ajudar os usuários a avaliarem seus próprios hábitos online. O objetivo é promover um uso mais equilibrado e saudável da tecnologia.

Projeto 3

Um site no formato de uma landing page, com um design inspirado nas roles de Overwatch. O objetivo do projeto foi oferecer uma experiência visual imersiva e interativa para os fãs do jogo, destacando as diferentes funções dos heróis e como elas contribuem para o jogo de equipe. Cada seção da página foi projetada para refletir as características únicas das roles de Overwatch — Tanque, Dano e Suporte — com uma abordagem visual que remete diretamente ao estilo dinâmico e estratégico do jogo.
